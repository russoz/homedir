OpenWorld is an icon theme for GNOME or KDE, which aims to 
follow FreeDesktop icon theme spec:
http://www.freedesktop.org/wiki/Standards/icon-theme-spec
 
This icons collection was built over couple of years, made for 
various purposes: StarOffice presentations, illustrations, 
design documents, OpenSolaris.org website, my blog etc.
Introduction and design principle is here:
http://blogs.sun.com/chandan?entry=simple_correct_vector_icons_and

All icons are original and are released under CDDL open source license.
http://www.sun.com/cddl/
